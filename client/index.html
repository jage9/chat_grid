Chat Grid

Help

Movement

Arrow Keys: Move

C: Coordinates

Escape: Disconnect/cancel

Users, Nickname, and Chat

L: Locate nearest user

Shift+L: List users

Shift+U: List connected users

N: Change nickname

Apostrophe: Start chat

Comma / Period: Previous/next chat message

Less Than / Greater Than: First/last chat message

Items

I: Locate nearest item

Shift+I: List items

A: Add item

O: Edit item properties

D: Pick up/drop item

Shift+D: Delete item

U: Use item

Audio

P: Ping server

M: Mute/unmute

Shift+M: Toggle stereo/mono output

E: Cycle voice effect

Dash or Equals: Lower/raise active effect value
